# Contributing to the TWIM bot config

Thank you for taking the time to contribute to Matrix!

This is the repository holding the configuration files for our TWIM bot,
an instance of [harharlinks/hebbot](https://github.com/harharlinks/hebbot),
forked from [haecker-felix/hebbot](https://github.com/haecker-felix/hebbot)
to add features specific for TWIM.

Learn more about TWIM at <https://matrix.org/twim-guide/>.

## Processes

The processes for development on this repository are inherited from the main matrix.org website repo
and can be found at <https://github.com/matrix-org/matrix.org/blob/main/CONTRIBUTING.md>.
